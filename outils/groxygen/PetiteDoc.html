présentables tout le fatras de données que l'équipe uchronia project produira chaque jour.
Evidemment, le nom évoque un peut doxygen, normal, j'ai trouvée la synthaxe de mon

Utilisation

Installation

Cette documentation est normalement fournie avec les sources de Groxygen, qui se
compile en executant basiquement le script compile.sh, en resortira un executable
nommé simplement groxygen, qui contient toutes les fonctionnalités citées ci-après.

Synthaxe

La synthaxe de doxygen est un melange douteux entre le c et le html, mefiez vous de ceux
qui invoquent ce genre de mariage incestueux.

Parce qu'il faut bien commencer par quelque chose, le fichier donné en paramètre est un
pur fichier de script, il sert à donner le cemin des autres fichiers qui composent la
documentation et à définir leur fonction dans leur documentation, ceci peut influer sur
la manière dont ceux-ci seront affichés. Ci-dessous les différentes balises que vous
pourrez déposer dans nimporte quel ordre dans ce fichier (faites en sorte que ça reste
lisible, hein, sinon, moi je sert à rien ;) .

  • mainpage: [chemin]

    • La page qui devra s'ouvrir en premier dans le site, elle bénéficie d'améliorations synthaxiques.
    que je détaillerais plus loin.

  • page: [chemin]

    • Juste une page de doc

  • savedir: [chemin]

    • On va donner l'endroit où seront sauvegardés les pages html compilés.

  • cmd: [tempo]; [commande]

    • Executera cette commande (système) au moment tempo définit. tempo peut prendre ces valeurs: BEGIN et END.
