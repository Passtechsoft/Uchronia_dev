Midpoint displacement noise, ou Diamond square noise (nom latin: carrus diamus)

Cet algo génère sur une zone memoire donnée un bruit de type midpoint displacement

Note
Vous trouverez des informations sur le fonctionnement interne de cette fonction dans le fichier algoinfos rubrique diamondSquareNoise
Warning
Il est interdit de mettre pos à 1 ou à 0, car ça signifierait que le nombre de point par coté vaut 2
Parameters
xLa position du chunk virtuelle X, inutile dans les versions primitives de la fonction
yLa position du chunk virtuelle Y, inutile dans les versions primitives de la fonction
amplitudeLa hauteur maximale d'un point (la valeur minimale est l'opposée de l'amplitude
resLa résolution, c'est à dire la rapidité de la réduction du bruit
subDivisionsLe nombre de subdivisions appliqués, pour trouver le nombre de points par coté ça fait, calculez: pow(2, subDivisions)+1
